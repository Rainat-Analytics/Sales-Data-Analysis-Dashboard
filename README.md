# Sales Data Analysis Dashboard (Excel)

## 📊 Project Overview
This project analyzes sales data to uncover trends, evaluate business performance, and support decision-making.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning Techniques
- Data Visualization (Charts & Dashboard)

## 🔍 Key Analysis Performed
- Sales trends over time
- Revenue by region
- Profit by category
- Top customers analysis
- Profit margin evaluation

## 📈 Dashboard
![Dashboard](dashboard.png)

## 💡 Key Insights
- Sales peak in Q4, indicating seasonal demand
- West region generates the highest revenue
- Technology category has the highest profit margin
- A small group of customers drives a large portion of revenue
